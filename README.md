## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/itzsudipta/itzsudipta.github.io.git
cd itzsudipta.github.io
```

2. **Install dependencies**
```bash
npm install
```

3. **Build Tailwind CSS**
```bash
npm run build
```

4. **Start local development server**
```bash
python -m http.server 8000
# or
npx serve
```

5. **Visit** `http://localhost:8000`
