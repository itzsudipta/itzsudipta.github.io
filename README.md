# Sudipta Sarkar | Portfolio

A modern, responsive personal portfolio website for Sudipta Sarkar, featuring a tech-themed design, project showcase, introduction, and contact section.

## Features
- Clean, tech-inspired layout
- Responsive design for all devices
- Project showcase with technologies used and GitHub links
- Contact form (placeholder)
- Fast loading and optimized performance

## How to Use
1. Open `index.html` in your browser to view the portfolio.
2. Update the introduction and project sections in `index.html` as needed.
3. Replace the professional picture URL with your actual image if desired.
4. Integrate the contact form with your preferred backend or email service.

## Customization
- Edit `src/styles.css` for color palette, fonts, and layout changes.
- Add or update projects in the Projects section of `index.html`.
